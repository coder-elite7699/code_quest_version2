Iterative Binary Search

You are given a sorted array of integers and a number K. Your task is to find wether the given number K exist in the given array or not. If the given number is present in the array, return the index at which it is present and if not present return -1. You have to write iterative code.

Example1:

arr = [1,4,11,19,25,36,39,40,42]

target = 19

output: 3

Explanation: The number 19 is present at index 3

Example2:

arr = [1,4,11,19,25,36,39,40,42]

target = 20

output: -1

Explanation: 20 is not present in given array

Explanation

  • we will keep to pointer , low and high
  • low is initialize with 0, low=0
  • high is initialized with n-1, where n is the size of the array, high=n-1, which will be pointing to last element of the array
  • we will calculate mid = (low+high)/2
  • check wether arr[mid]==target
  • There arise three possibility:
    • either the target will be smaller than the mid element of the array
    • or it will be greater than the mid element of the array
    • or it will be equal to the mid element of the array
  • If target element is greater than mid element then shift the low pointer to mid+1, ot next to mid element
  • If target element is smaller than mid element then shift the high pointer to mid-1, or previous to mid element
  • if target==array[mid] the return mide
  • Continue this till low crosses high or vice versa
