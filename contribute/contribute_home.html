How can you contribute to this website?

There are major 5 field you can contribute to this Website.

  • Coding Problem: You can contribute to coding problme listed, by writing problem description, its code and explanation.
  • Aptitude Section: Add new question that has been asked in an interview. You need to show proof of the question
  • Core CS: In this section you can write article on Operating system, Database Management System(DBMS) , Computer Network and Object Oriented Programming

What you should not do.

  • Use of any LLM(Large Language Model) like ChatGPT, Gemini, Claude etc to write article, Coding solution
  • Copying some other articles of other webiste present on internet
  • Correction of spelling mistakes in readME file or in webiste , just for sake of Contribution
