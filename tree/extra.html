Recursive PreOrder Traversal

You are given a binary tree. Your task is to return an array that conatins the Preorder Traversal. A preorder Traversal is Root->Left->Right

Example1:

Output: 1 2 3 4 8 9 7

Explanation

  • Whenever you encounter a node print it.
  • Then check if left of current node exist if exist then move to that node and print it.
  • If at any moment left node doesnot exist the move to right and print it and repeat above step
