Recursive Post order Traversal

You are given a binary tree. Your task is to return an array that contains the Postorder Traversal. A Post order Traversal is Left->Right->Root

Example1:

Output: 3 2 4 1 5 7 9

Explanation

Algorithm:

  • Visit Left Subtree: Go as far to the left as possible, visiting each node in the left subtree.
  • Then Visit Current Node: Once you reach a leaf node (a node with no children), move to the current node and visit it.
  • Finally, Visit Right Subtree: After visiting the current node, go as far to the right as possible, visiting each node in the right subtree.
