Largest Element

Your are given an array of positive integers. Your task is to find the largest element in the given array

Example1:

arr = [78, 96, 14, 82, 35, 74, 6, 85, 2, 63, 64, 25]

output: 96

Explanation: The number 85 is largest in array

Example2:

arr = [45, 54, 62, 16, 25, 12, 65, 64, 52, 59, 44, 23]

output: 65

Explanation: 65 is largest element in the given array

Explanation

  • Initialize max = arr[0].
  • Iterate through the array starting from the second element.
  • For each element, check if arr[i] > max.
  • If true, update max to arr[i].
  • After the loop ends, max will hold the largest element.
  • Return max.
